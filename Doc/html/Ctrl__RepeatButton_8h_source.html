00026 // CRepeatButton is just an ordinary button but if you hold the Left Mouse

00035 // sound during the auto-repeat to give further indication to the user that
00036 // something non-standard is happening.
00037 //
00038 // The trick to using this control is to use an ordinary button control on your

00040 // Category: Control, Variable type: CRepeatButton
