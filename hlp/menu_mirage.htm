Mirage commands

The Mirage menu offers the following commands:

With the exception of Preferences the Mirage must be booted with MASOS in order for these options to work.
Receive Sample Get sample(s) from the Mirage.
Transmit Sample Send sample to the Mirage.
Send Looppoints Send only the parameters for the loop.
Key Mapping Edit key mapping for the samples.
Preferences Select the MIDI ports for communication with the Mirage.
