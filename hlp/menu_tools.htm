Tools menu

The Tools menu offers the following commands:

Loop Editor Used to edit the looppoints in the active sample.
Switching the loop off and creating an alternate loop
is also done here.
Reverse Simple reverses the active sample.
Resample Resamples the sample to a higher or lower samplerate,
usefull to fit the sample in available memory.
Normalize Normalize the sample to 100% amplitude.
Resynthesize Resynthesize the loop.
