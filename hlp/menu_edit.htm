Edit menu

The Edit menu offers the following commands:

Undo Reverse previous editing operation.
Redo Reverse the previous undo
Cut Cuts the selected part from the wave
Copy Copies the selected part of the wave
Paste Paste data from the clipboard, overwriting the current selection.
Paste Special
Append Paste the clipboard after the current selection.
Prepend Paste the clipboard before the current selection.
Multiple Copies Paste multiple copies of the clipboard in the current selection.
Mix Mix the clipboard with the current selection.
Truncate  
Before Loop Truncate data before the loop start
After Loop Truncate data after loop end
Keep Loop Only Keep only the loop
