File menu commands

The File menu offers the following commands:

Open Opens an existing file.
Files may be mono or stereo, 8 or 16 bits.
Conversion to 8 bits mono is automatically done
Wavefiles larger than 64 kb (after conversion to mono/8 bits)
are resampled to fit in 64 kb.
Close Closes an opened Wavesample.
Save Saves an opened Wavefile using the same file name.
Save As Saves an opened Wavefile to a specified file name.
Exit Exits Mirage Editor.
